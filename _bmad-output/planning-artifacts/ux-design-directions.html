Bosco — Design Directions

3 directions pour le dashboard (l'écran le plus critique), plus les écrans communs Chat et Journal.

Contexte

Le dashboard est l'écran que le capitaine voit chaque matin. C'est LE moment qui définit le produit. Les 3 directions explorent différents équilibres entre impact émotionnel et densité d'information.

  • Direction A "Verdict Immersif" — Le verdict domine. Maximum d'impact émotionnel, minimum de distraction. L'info secondaire est sous le fold.
  • Direction B "Command Center" — Le verdict est compact, entouré de données. Plus dense, plus informatif, moins émotionnel. Comme un tableau de bord d'instruments.
  • Direction C "Card Stack" — Chaque bloc d'info est une carte autonome dans un flux vertical. Verdict en premier, puis météo, puis route, etc. Style moderne "mobile-native".

Pourquoi cette direction est recommandée

  • Le verdict est visible en < 2 secondes, lisible à 2 mètres — correspond exactement au "10-second rule"
  • Le gradient de couleur communique l'émotion avant même de lire le texte
  • L'info secondaire (météo, niveaux, route) est accessible par scroll, pas en compétition visuelle
  • Pattern Apple Weather prouvé : hero data + progressive scroll
  • Supporte naturellement les 3 états émotionnels : confiance (GO), prudence (STANDBY), acceptation (NO-GO)

Forces et faiblesses

  • ✅ Plus d'info visible sans scroller — pratique au mouillage quand on a le temps
  • ✅ Grid 2x2 donne un aperçu complet des métriques
  • ❌ Le verdict est compressé — perd l'impact émotionnel du "hero moment"
  • ❌ Trop dense pour le cockpit — charge cognitive élevée avec 6+ data points visibles
  • ❌ Ressemble plus à un dashboard technique qu'à un "second de bord"

Design chat — Points clés

  • Les ActionCards remplacent les badges bruts du POC — résumés humains avec icônes de statut
  • Les suggestion chips sont contextuels (changent selon la dernière conversation/position)
  • Le texte de l'IA est conversationnel, utilise le tutoiement, fait référence au bateau par son nom
  • Les actions multiples sont groupées visuellement, pas éclatées en badges individuels

Recommandation

Direction A "Verdict Immersif" est la meilleure correspondance avec nos principes. Elle maximise le "hero moment" du verdict tout en offrant la progressive disclosure nécessaire pour les détails. C'est Apple Weather appliqué à la navigation.

On pourrait emprunter à la Direction C le style de cartes arrondies (border-radius: 16px) et la météo en grid 3 colonnes pour les sections sous le fold.
